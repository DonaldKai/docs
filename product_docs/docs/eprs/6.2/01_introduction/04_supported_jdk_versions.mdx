| Debian 10                               | Red Hat OpenJDK 11                                    |
| Ubuntu 18.04 and 20.04                  | Red Hat OpenJDK 11                                    |
| Windows 2012 R2, 2016, and 2019         | Red Hat OpenJDK 8                                     |
